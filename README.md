# Simple SUAP

## Objetivo

Esse projeto tem como objetivo criar uma interface simples e responsiva para visualização das informações do módulo ensino do [SUAP](https://portal.suap.ifrn.edu.br/).

Dentre as principais informações estão:

- Dados principais do discente logado;

- Horários e locais de aula; 

- Boletins de todos os semetres.

Para o futuro, será necessário um sitema simples para autenticação OAuth com o SUAP e envios dos dados para a interface.

## Tecnologias

A principal tecnologia utilizada para a implementação da interface será o framework [Angular](https://angular.dev/)
